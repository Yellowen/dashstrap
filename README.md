# Dashstrap

Dashstrap is a dashboard interface for [Faalis](http://github.com/Yellowen/Faalis) which is using **AdminLTE**.
## Installation

Add this line to your application's Gemfile:

    gem 'dashstrap'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dashstrap
